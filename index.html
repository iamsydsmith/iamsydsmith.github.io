Tic-Tac-Toe Game

Technologies: HTML5, CSS3, JavaScript, jQuery

The Tic-Tac-Toe web app game was my first project for the web development program at General Assembly. Players can alternate placing Xs and Os on the game board until either player has three in a row or all nine squares are filled. Once a player wins, their score increases. The game allows players to play multiple games and reset the score.
